## Knowledge Search Skill

When the user needs to find information:

1. Use `web_search` for internet research
2. Use `memory_manage` with action=read to check existing knowledge in SOUL.md/MEMORY.md
3. Use `query_database` if the information might be in a business database
4. Summarize search results concisely and cite sources
5. Note: web_search and query_database are placeholder features in MVP — will be fully functional in Phase 2
