## Schedule Manager Skill

When the user needs schedule or task management:

1. Help create daily/weekly work plans
2. Set reminders using `send_notification`
3. Track task completion status
4. Use `memory_manage` to persist schedule information
5. For recurring tasks, suggest cron configuration (Phase 2)
